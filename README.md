# EnvMonBT
[Software](https://github.com/KimiakiK/EnvMonBT-soft)

# Features
- 温度湿度気圧CO2計
- 直近1時間と24時間のグラフ表示
- 統計表示(平均、最大、最小)
- BLE接続

# Major Components  
- STM32F303K8
- 温湿度・気圧センサ BME280
- CO2センサ MH-Z19
- TFT 1.3インチ 240x240ドット
- Bluetooth RN4020
